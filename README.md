# Tema 6 · Archivo

Repositorio didáctico preparado para GitHub Pages.

## Publicación rápida
1. Sube **todo** el contenido del repositorio a GitHub.
2. Mantén `index.html` en la raíz.
3. Activa GitHub Pages desde la rama principal.

## Funciones incluidas
- 7 fases
- mapa conceptual con huecos autocorregibles
- test del mapa (5 preguntas aleatorias)
- práctica por fase
- código de verificación docente oculto (Alt + Shift + D)
- test aleatorio de 10 preguntas por fase
- máximo 2 intentos
- corrección mostrada al finalizar
- progreso guardado en localStorage
- certificado final imprimible
